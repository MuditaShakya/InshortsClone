# Inshorts Clone

A MERN stack news application inspired by Inshorts.

## Features
- Fetches latest news from backend API
- Infinite scrolling
- Clean and responsive UI
- REST API integration

## Tech Stack
Frontend: React, JavaScript, HTML, CSS  
Backend: Node.js, Express  
Database: MongoDB
